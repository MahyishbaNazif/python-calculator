# python-calculator
# 🧮 Python Calculator

A menu-driven console calculator built in Python.

## Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧮 Modulus
- 📐 Floor Division
- ⚡ Power
- √ Square Root
- ❌ Division by Zero Handling
- 🔁 Continuous Calculations
- 🚪 Exit Option

## Technologies Used

- Python 3
- Functions
- Loops
- Conditional Statements
- Exception Handling
- Math Module

## Project Structure

```
python-calculator/
│
├── calculator.py
└── README.md
```

## How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project

```bash
cd python-calculator
```

3. Run the program

```bash
python calculator.py
```
